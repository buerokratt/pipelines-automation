# Pipelines Automation

<details>
<summary><b>Commit Message Validation</b></summary>

- Install Husky:
  ```bash
  npm install husky
  ```

- Initialize Husky:
  ```bash
  npx husky init
  ```
- Copy commit-msg to your .husky directory.
</details>

<details>
<summary><b>Automatic Changelog Generation</b></summary>

- Generate Changelog:
  ```bash
  npm run changelog
  ```
</details>
